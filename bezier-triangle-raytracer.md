The aim of this project is to simulate image rendering of arbitrary-shaped "lenses" described by triangle meshes using Bézier triangle interpolation. It will be possible to simulate free-form lenses, and of course spherical lenses as well.

[github.com/balazs-bamer/bezier-triangle-raytracer](https://github.com/balazs-bamer/bezier-triangle-raytracer)
